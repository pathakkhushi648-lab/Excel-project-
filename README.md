 Excel-Car Sales Data Analysis & Dashboard Project

  Project Overview

This project focuses on analyzing car sales data and building an
interactive dashboard using Microsoft Excel. The goal is to transform
raw data into meaningful business insights using Pivot Tables, Charts,
and KPIs.


 Objectives

-   Clean and organize raw sales data
-   Perform data analysis using Excel formulas
-   Create Pivot Tables for summarization
-   Design an interactive dashboard
-   Generate business insights from the dataset

 Tools & Technologies Used

-   Microsoft Excel
-   Pivot Tables
-   Pivot Charts
-   Excel Functions (SUM, AVERAGE, COUNT, IF, etc.)
-   Data Cleaning Techniques


 Project Structure

Project Folder - dataset.xlsx
- dashboard.xlsx
- screenshots
- README.md


 Dashboard Features

-   Sales Trend Analysis
-   City-wise Performance
-   Brand-wise Comparison
-   KPI Cards (Total Revenue, Total Cars Sold, Average Price)
-   Interactive Filters & Slicers


 Key Insights

-   Identified highest revenue-generating car brands
-   Analyzed monthly sales growth trends
-   Compared performance across different cities
-   Found top-selling car models
